Specify whether to generate the deprecated API page (deprecated-list.html), which contains the list of deprecated APIs, and the "Deprecated" link in the navigation bar to that page.

Note: This parameter has no effect on the generation of the deprecated API throughout the rest of the documentation, which is controlled separately by "Include | Deprecated API" parameter.

Default Value:

The default value of this parameter is set according to the -nodeprecatedlist option found on the Javadoc command line. When this option is specified, the default value is false, otherwise it is true.
